Privacy Policy

Your privacy matters. LearnQuest is designed with privacy at its core.

Effective Date: February 1, 2026

LearnQuest collects zero personal data. The application runs entirely offline on your local device. No information is transmitted to any server, ever.

1. Overview

LearnQuest is a free, open-source, offline educational platform designed for students in grades K-8. This privacy policy explains how LearnQuest handles (or rather, does not handle) user data. We are committed to protecting the privacy of students, teachers, and all users.

2. Data Collection

We do not collect any personal data. LearnQuest operates entirely offline after the initial setup. Specifically:

  • No user accounts are transmitted to any server
  • No browsing or usage data is collected
  • No cookies are sent to third parties
  • No analytics or tracking tools are used
  • No advertising is displayed
  • No data is shared with any third party

3. Local Data Storage

All data created by LearnQuest (student profiles, progress, quiz scores, chat history) is stored locally in a SQLite database on the device or USB drive where LearnQuest is installed. This data:

  • Never leaves the local device
  • Is fully under the control of the user or administrator
  • Can be deleted at any time by removing the database file
  • Is not encrypted by default (physical device security is the responsibility of the user)

4. AI Model

LearnQuest uses a locally-running AI model (Phi-3 Mini via Ollama) for tutoring features. This model:

  • Runs entirely on the local machine
  • Does not send any queries or responses to external servers
  • Does not learn from or retain student interactions between sessions
  • Requires no internet connection to operate

5. Initial Setup

Internet connectivity is required only during the initial setup to download:

  • The Ollama runtime (if not already installed)
  • The Phi-3 Mini AI model
  • Node.js package dependencies

After setup is complete, no internet connection is needed or used.

6. COPPA Compliance

Because LearnQuest does not collect, store, or transmit any personal information to external servers, it is designed to be compatible with the Children's Online Privacy Protection Act (COPPA). No parental consent mechanism is needed because no data collection occurs.

7. FERPA Compliance

LearnQuest does not create, maintain, or transmit education records to any external party. All student progress data remains on the local device under the control of the school or user, supporting compliance with the Family Educational Rights and Privacy Act (FERPA).

8. Open Source Verification

LearnQuest is fully open source under the MIT License. Anyone can inspect the source code to verify that no data collection or transmission occurs. Our code is publicly available at github.com/edwardxiong2027/LearnQuest-Node.

9. This Website

This showcase website (learnquest2026.web.app) is hosted on Firebase Hosting. Firebase may collect standard web server logs (IP addresses, browser type, pages visited) as part of its hosting infrastructure. This website does not use any additional analytics, tracking cookies, or advertising. The LearnQuest application itself, which runs offline, is entirely separate from this website.

10. Changes to This Policy

If we make changes to this privacy policy, we will update the effective date at the top of this page and note the changes in our GitHub repository. Because LearnQuest does not collect user data, we have no way to notify users directly of policy changes.
