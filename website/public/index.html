How It Works

Get up and running in three simple steps. No technical expertise required.

1

Download

Clone the repository or copy the files to a USB drive. Everything you need is included.

2

Setup

Run the setup wizard to install dependencies and download the AI model. Takes about 5 minutes.

3

Learn

Open your browser and start learning! Pick a subject, choose your grade, and dive in.

Built by Students, for Students

We believe that every child deserves access to a patient, knowledgeable tutor that is available anytime, anywhere.
