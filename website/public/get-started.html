Get Started

Set up LearnQuest in minutes. Choose your platform and follow the guide below.

System Requirements

✓ Mac, Windows, or Linux
✓ Node.js 18 or higher
✓ 4 GB RAM minimum (8 GB recommended)
✓ 5 GB free disk space
✓ Any modern web browser
✓ Internet for initial setup only

Installation Guide

Internet is only needed for the initial setup. After that, everything runs offline.

Install on Windows

1

Install Node.js

Download Node.js v18+ (LTS) from nodejs.org. Run the installer with default settings.

2

Install Ollama

Download the Windows installer from ollama.com/download and run it.

3

Download LearnQuest

Go to GitHub Releases and download the source code ZIP, or clone with git.

Extract the ZIP to any folder (e.g., C:\LearnQuest or a USB drive).

4

Run Setup & Start

Open Command Prompt or PowerShell, navigate to the LearnQuest folder, and run:

# First-time setup (downloads AI model, installs dependencies) scripts\setup.bat # Start LearnQuest (opens browser automatically) scripts\start.bat

Or run npm install then npm start.

Install on macOS

1

Install Node.js

Install Node.js v18+ from nodejs.org, or via Homebrew: brew install node

2

Install Ollama

Download Ollama for Mac from ollama.com/download. Open the .dmg and drag Ollama to Applications.

3

Download LearnQuest

Download from GitHub Releases or clone with git:

git clone https://github.com/edwardxiong2027/LearnQuest-Node.git
4

Run Setup & Start

Open Terminal, navigate to the LearnQuest-Node folder, and run:

# First-time setup chmod +x scripts/setup.sh && ./scripts/setup.sh # Start LearnQuest ./scripts/start.sh

Or run npm install then npm start

Install on Linux / Chromebook

1

Install Node.js & Git

# Ubuntu / Debian / Chromebook (Crostini) sudo apt update && sudo apt install nodejs npm git -y
2

Install Ollama

curl -fsSL https://ollama.com/install.sh | sh
3

Clone & Setup

git clone https://github.com/edwardxiong2027/LearnQuest-Node.git cd LearnQuest-Node chmod +x scripts/setup.sh && ./scripts/setup.sh
4

Start Learning

./scripts/start.sh

Chromebook note: You need to enable Linux (Crostini) in Settings → Advanced → Developers first.

Install from GitHub (Any Platform)

1

Prerequisites

Install Node.js v18+ and Ollama for your platform.

2

Clone, Setup, and Start

# Clone the repository git clone https://github.com/edwardxiong2027/LearnQuest-Node.git cd LearnQuest-Node # Install dependencies npm install # Download the AI model (one-time, ~2.3 GB) ollama pull phi3 # Start LearnQuest npm start
3

Or Download ZIP

Don't have git? Go to github.com/edwardxiong2027/LearnQuest-Node, click the green "Code" button, then "Download ZIP". Extract and follow the same setup steps.
